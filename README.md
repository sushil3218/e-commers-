E-Commerce Web Application

This project is a modern e-commerce platform that allows users to browse products, search by categories, add items to a shopping cart, and place orders online. The system includes user authentication, product management, responsive design, and AI-powered product recommendations to enhance the shopping experience.

Features:
• User Registration & Login
• Product Search and Filtering
• Category Management
• Shopping Cart
• Order Management
• Responsive UI Design
